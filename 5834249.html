Dropdowns

La clase .dropdown se usa para indicar un menú desplegable.

Use la clase .dropdown-menu para contruir el menú desplegable.

Para abrir el menu desplegable, use el botón o un enlace con una clase de .dropdown-toggle y data-toggle ="dropdown".
